# 📁 Custom Virtual File System (VFS) - C++

## 🚀 Overview

A custom Virtual File System (VFS) implemented using **C-style system programming concepts in C++**, designed to simulate the core functionality of real-world file systems. This project demonstrates low-level file handling, memory management, and directory structure operations inspired by UNIX-based file systems.

---

## ✨ Features

* 📄 Create, delete, read, and write files
* 📂 Directory structure management
* 🧠 In-memory file system implementation
* 🔐 Permission handling (if implemented)
* ⚙️ Command-based interface for operations
* 🗂️ File metadata handling (size, type, etc.)

---

## 🛠️ Tech Stack

* **Language:** C (C-style) in C++ (.cpp)
* **Concepts Used:**

  * File System Design
  * Memory Management
  * Data Structures (Linked List / Arrays)
  * System Programming

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/vivekbgautam/custom-vfs-cpp.git
```

2. Navigate to the project folder:

```bash
cd custom-vfs-cpp
```

3. Compile the code:

```bash
g++ main.cpp -o vfs
```

4. Run the program:

```bash
./vfs
```
